IBM PC and MS-DOS character sets
--------------------------------

This file does not contain the Japanese CP932 "Shift-JIS" character set
because it is a multibyte character set with 7518 code points instead of 256.
The complete CP932 documentation is available in a separate
"CP932 Charmap.txt" file.
